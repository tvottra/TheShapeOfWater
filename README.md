# TheShapeOfWater
By Jonathan L, Victor L, Tommy V. T.

**Objective**: Make a calculator that computes the 3D critical points of a substance, as well as other properties of the substance, and finally generating a 3D surface plot of that substance.

##
*How To Run*

Add the three .jar files from /lib to your external jar library in Eclipse or to your Project Structure > Libraries in Intellij.
Run TheShapeOfWaterDriver.java
